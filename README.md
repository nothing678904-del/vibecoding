# ? Gas Price Mood

> Feel the Ethereum Network - Don't Calculate It

Immersive Next.js app that maps Ethereum gas data to an emotional network stress experience.

## Setup
1. Install Node.js 20+
2. `npm install`
3. `npm run dev`
4. Open http://localhost:3000

## Env
Copy `.env.example` to `.env.local` and set your values.
